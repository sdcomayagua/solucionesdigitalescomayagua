# SDC Smart Panel Extremo V4

Sistema privado para SD Comayagua con inventario, catálogo, cotización, ventas, recibos, clientes y administración.

## Cambios V4

- El botón Detalle fue corregido.
- Ahora al tocar cualquier parte de la tarjeta del producto se abre el modal de detalle.
- El modal de producto incluye botones: COTIZAR, VENDER y WHATSAPP.
- Se mejoró el formato del mensaje de WhatsApp para que los productos, cantidades, subtotales y totales se vean separados.
- Se agregaron botones independientes en Cotizar:
  - WhatsApp Comayagua, sin envío.
  - WhatsApp envío normal, productos + Lps. 110.
  - WhatsApp pagar al recibir, productos + Lps. 110 + 10% sobre productos + envío.
  - WhatsApp ambos, mostrando las dos opciones para clientes fuera de Comayagua.

## Uso

Abrir `index.html` o subir la carpeta completa a GitHub Pages.

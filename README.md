# SDC Smart Panel

Versión rediseñada desde cero del sistema de inventario para SD Comayagua.

## Incluye

- Inventario local con `localStorage`.
- Catálogo por categorías.
- Cotizaciones para WhatsApp.
- Registro de ventas con validación de stock.
- Historial de recibos.
- Descarga de recibo como imagen PNG.
- Clientes con historial básico.
- Administración de productos.
- Importar/exportar productos JSON o CSV.
- Diseño responsive para celular y escritorio.

## Cómo usar

Abre `index.html` directamente o sube todos los archivos a GitHub Pages.

Los productos iniciales se cargan desde `products_from_xlsx.json`. Los cambios que hagas desde Admin quedan guardados en el navegador.

## Archivos principales

- `index.html`
- `css/app.css`
- `js/app.js`
- `products_from_xlsx.json`
- `manifest.webmanifest`
- `assets/`
